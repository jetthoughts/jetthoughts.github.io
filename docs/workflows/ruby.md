# Ruby/Rails Flow

Use this flow for any Ruby/Rails changes.

1. Read `@docs/workflows/BASE_HANDBOOK.md`
2. Use skills: `37signals-rails`, `rails-expert`, `minitest`
3. Run `bin/rake test:critical` after every micro-change
4. For HTML/CSS changes triggered by Rails views, also read `@docs/workflows/css-consolidation.md`
