# Blog Pipeline (9 Phases)

Canonical workflow for blog creation and publishing.

1. Trend Discovery (`social-media-trends-research` + `keyword-research`)
2. Strategy (`content-strategy` + `content-calendar-planner`)
3. Research (`competitor-intel` + `web-research-workflow`)
4. Ideation (`blog-post` + `copywriting-core`)
5. Draft & Edit (`blog-writer` + `content-production` + `copy-editing`)
6. AI Quality Check (`slop-detector` → `humanizer` if flagged)
7. SEO & Performance (`seo` + `seo-aeo-audit` + `pagespeed-insights`)
8. Cover Image (if needed) → `@docs/workflows/cover-images.md` and `@.stitch/design.md`
9. Hugo Build (`hugo` + `obsidian-markdown`) → `bin/hugo-build`
10. Continuous Improvement (`self-improving-agent`)
