# LinkedIn Course Promotion Plan — "From Idea to First Paying Customer"

**Purpose:** Weekly LinkedIn promotion of the free course at `/course/tech-for-non-technical-founders-2026/`, posted as Paul Keen.
**Cadence:** Up to 5 posts/week, one course module per week, 5-week core cycle, then evergreen rotation.
**Voice:** All rules in `docs/workflows/linkedin-post-pipeline.md` apply verbatim (story shape, 5 sentence tests, AI score ≤ 2/10, opener rotation, no em dashes). This file only adds the course-promo layer.
**Save location for drafts:** `linkedin-posts/course-promo/week{N}-{day}-{slug}.md` (same frontmatter format as ICP campaign, plus `campaign: course-promo`).

---

## What we are promoting (facts the posts can use)

- Free five-module course, no sign-up, no email gate, evenings-and-weekends pace
- Route: idea → Founding Hypothesis → smoke test with 300 cold visitors → 10 Mom Test interviews → one-page brief → self-built MVP (Lovable + Supabase + Stripe) → signed paid pilot
- Every module ends in an artifact; the artifacts bundle into a "Founder OS" folder
- Go/iterate/kill verdict on the idea within 2-3 weeks, before any money is spent

Never state these as a feature list in a post. Each post gives away ONE piece of the course as standalone value; the course link is the "there's more where this came from" afterthought.

## Link policy (amendment to the pipeline, decided 2026-07-12)

The pipeline bans website links and JT mentions in Paul's post bodies. For this campaign:

- **Post body:** still zero links, zero "JetThoughts", zero "check out my course" phrasing. The body must deliver its full value without the click.
- **First comment (Paul's own, posted immediately):** one plain sentence + course URL with UTM. Example: `The full playbook is free, no email gate: <url>?utm_source=linkedin&utm_medium=social&utm_campaign=course_promo_module1&utm_content=<slug>`
- **Exception - Friday pointer post:** one link allowed in the body, because the post's honest subject IS the course ("I wrote the whole thing down"). One Friday post per week maximum uses this.

Rationale: the course is a genuine give-away (free, ungated), so sharing it is closer to the pipeline's "direct give-away" rule than to lead-gen. Same trade-off style as the H5 decision in the ICP plan.

## The weekly rhythm (up to 5 slots, drop from the bottom if fewer)

Every week covers one module. Same skeleton every week so production stays cheap and readers learn the rhythm:

| Slot | Shape | Design attachment | Purpose |
|---|---|---|---|
| Mon | Personal story post | **None** (plain text) or Format A | The mistake this module prevents, told through one real encounter |
| Tue | Tactical give-away | Format B (napkin sketch) | One complete, usable technique from the module, fully in the post |
| Wed | Poll | Native poll (no image) | Cheap engagement + ICP signal on the module's pain |
| Thu | Artifact show-and-tell | Format A (marked screenshot) | Show a real filled-in course template; proof, not promise |
| Fri | Course pointer | Format A (screenshot of course page) | "This week's posts came from Module N, the rest is here" + link |

Priority when cutting to 3 posts/week: keep Mon, Tue, Fri.

## Design system for attachments (simplest first, most personal first)

Four repeatable formats. No polished corporate graphics, no stock imagery, no carousels in the core cycle.

| Format | What it is | How to produce | When |
|---|---|---|---|
| **A. Marked screenshot** | Real screenshot (course page, filled template, Lovable build, Stripe test payment) with ONE hand-drawn red circle or arrow, nothing else | Screenshot + Excalidraw red pen (`#cc342d`) | Thu artifact posts, Fri pointers |
| **B. Napkin sketch** | One idea drawn by hand: a fork, a formula, a 5-stop route. Max 15 words inside the sketch | Excalidraw, hand-drawn style, black on white, one red accent | Tue give-aways |
| **C. One-line text card** | A single conviction sentence, dark background, white text, red underline accent | 1200x1200, JetVelocity obsidian bg | Sparingly - max 1/week, only when no honest screenshot exists |
| **D. Nothing** | Plain text post | - | Mon stories - bare personal posts routinely outperform decorated ones |

Rules: labels live inside the sketch, never in a caption paragraph. If removing the image loses no information, post without it (Format D beats decoration). Every visual gets checked at phone width before scheduling - LinkedIn is read on phones.

Weekly Friday review, same table as the ICP plan. Decision rules:

- If Tue give-aways beat Fri pointers on course clicks, the value-first framing is working - do not add harder CTAs.

- If reader-result posts appear, they outrank everything in the next week's plan.
