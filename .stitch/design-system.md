# JetVelocity Design System — Blog Cover Images

## Overview
**Creative North Star**: "The Obsidian Engine"
High-end IDE/technical blueprint aesthetic — authoritative, precise, sophisticated dark atmosphere.

## Colors
| Token | Hex | Usage |
|-------|-----|-------|
| Background | `#10141a` | Primary canvas |
| Primary (Ruby Red) | `#cc342d` | Ruby/brand accents, badges, CTAs |
| Secondary (Neon Purple) | `#a855f7` | Gradient accents, highlights |
| Surface | `#10141a` | Base background |
| Surface Low | `#181c22` | Content blocks |
| Surface Container | `#1c2026` | Cards, code blocks |
| On Background | `#dfe2eb` | Body text (never pure white) |
| Tertiary (Cyan) | `#7dd0fe` | Accent lines, tech details |

## Typography
- **Headlines**: Space Grotesk — aggressive, technical authority
- **Body/Metadata**: Inter — clean readability
- **Labels**: Inter, ALL CAPS, tracked out (+0.05em)

## Layout Pattern (1200×630 OG Standard)
1. **Background**: Dark gradient (#10141a → #1a0a2e) with subtle geometric patterns
2. **Top-left**: "JetThoughts" brand mark
3. **Top-right**: Topic badge/tag pill (Ruby red `#cc342d`)
4. **Center-left**: Large bold headline (2-3 lines max, white `#dfe2eb`)
5. **Bottom-left**: Stat/feature badges in a row (colored tags)
6. **Right side**: Abstract visual element (glowing gem/crystal/icon in brand colors)

## Rules
- **No-Line Rule**: No 1px solid borders for sectioning — use tonal shifts
- **Glass & Gradient**: CTA gradient from `#cc342d` to `#6f00be` at 135°
- **Intentional Asymmetry**: Heavy left-aligned typography, negative space right
- **Never use #FFFFFF** — always `#dfe2eb` for text
- **Thumbnail readable**: Text must be legible at 300×157px
