# JetThoughts Blog - Design System

## Project Overview
- **Type**: Tech blog cover images and feature graphics
- **Platform**: Web (static site)
- **Style**: Professional, modern, developer-focused

## Design Tokens

### Color Palette
- **Primary**: Ruby Red (#DC143C) - Used for accents, highlights, CTAs
- **Secondary**: Deep Navy (#0F172A) - Backgrounds, headers
- **Accent**: Electric Blue (#3B82F6) - Links, interactive elements
- **Text Primary**: Slate 900 (#0F172A) - Headlines, body text
- **Text Secondary**: Slate 600 (#475569) - Subtitles, descriptions
- **Background**: White (#FFFFFF) - Card backgrounds
- **Surface**: Slate 50 (#F8FAFC) - Page backgrounds

### Typography
- **Headlines**: Bold, sans-serif (Inter/Roboto style)
- **Body**: Clean, readable sans-serif
- **Code**: Monospace (JetBrains Mono style for code snippets)

### Layout
- **Card Style**: Clean white cards with subtle shadows
- **Borders**: Rounded corners (8-12px)
- **Shadows**: Soft, diffused shadows for depth
- **Spacing**: Generous whitespace, clear hierarchy

## Visual Language
- **Vibe**: Professional, technical, modern
- **Mood**: Clean, minimal, developer-friendly
- **Atmosphere**: Trustworthy, authoritative, accessible

## Cover Image Specifications
- **Dimensions**: 1200x630px (Open Graph standard)
- **Format**: JPEG (progressive)
- **Style**: Bold typography, gradient backgrounds, code elements
- **Content**: Title, tags, abstract visual elements related to topic

## Component Patterns

### Blog Cover Image
1. **Header Area**: Blog name/logo (JetThoughts)
2. **Title Section**: Large, bold post title (2-3 lines max)
3. **Tag/Category Badge**: Topic tags in colored pills
4. **Visual Element**: Abstract shapes, code snippets, or icons
5. **Date/Author**: Small text at bottom

### Background Styles
- Gradient overlays (navy to dark blue)
- Subtle geometric patterns
- Code syntax highlighting as texture
- Abstract tech-related graphics
